# 🏠 House Price Prediction Web App

## 📌 Project Overview

This project is a **Machine Learning-based web application** that predicts house prices based on various input features.
It uses a trained ML model integrated with a Flask web interface to provide real-time predictions.

---

## 🚀 Features

* Predict house prices instantly
* User-friendly web interface
* Machine Learning model integration
* Scalable and easy to deploy

---

## 🛠️ Tech Stack

* **Python**
* **Flask**
* **Pandas**
* **NumPy**
* **Scikit-learn**
* **HTML / CSS**

## ▶️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/khatri174/Houseworth_prediction.git
cd Houseworth_prediction
```

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000/
```
