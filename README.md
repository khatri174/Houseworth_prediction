# 🏠 House Price Prediction Web App

## 📌 Project Overview

This project is a **Machine Learning-based web application** that predicts house prices based on various input features.
It uses a trained ML model integrated with a Flask web interface to provide real-time predictions.

---

## 🚀 Features

* Predict house prices instantly
* User-friendly web interface
* Machine Learning model integration
* Scalable and easy to deploy

---

## 🛠️ Tech Stack

* **Python**
* **Flask**
* **Pandas**
* **NumPy**
* **Scikit-learn**
* **HTML / CSS**
